# SeattleZaroni

A data-driven website ranking the best pizza places in Seattle.

## Prerequisites

- Node.js (v14 or higher)
- npm (Node Package Manager)

## Installation

1. Clone the repository or navigate to the project directory.
2. Install dependencies:

```bash
npm install
```

## Usage

Start the application:

```bash
npm start
```

The application will start on port 3000 (or the port specified in the `PORT` environment variable).
Open your browser and navigate to `http://localhost:3000`.

## Structure

- `server.js`: The main Express server file.
- `views/`: Contains EJS templates.
- `public/`: Contains static assets (images, CSS, JS).
- `package.json`: Project configuration and dependencies.

## Technologies

- Node.js
- Express.js
- EJS
- Tailwind CSS (via CDN)
